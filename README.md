# ACompany
: 회사 내부 사원들의 소통 & 연결을 독려하는 기능을 제공하는 웹 사이트입니다.

<br>

## 담당 기능
### 1. 카풀 (Carpool)
: 사원들간의 출/퇴근 카풀 목록을 등록/조회/신청/관리 할 수 있습니다
 
##### 구성
 카풀 번호, 날짜, 위치(주소, 좌표), 운전자, 차량, 탑승 인원, 거리, 도착시간 
 
##### 주요화면
- CarPool 목록 / 신청
- CarPool 등록
- 나의 CarPool
- 운행 기록 관리

##### 사용 API
- 카카오 지도 api : 위치 표시
- 카카오 모빌리티 api : 거리, 시간 계산

### 2. 마일리지샵 (MileageShop)
: 사내 마일리지를 사용해 사무용품을 구매할 수 있는 쇼핑몰 기능입니다.

##### 주요화면
- 상품 목록 조회
- 상품 상세정보 조회
- 장바구니
- 결제

##### 사용 API 및 라이브러리
- Selenium : 사무용품 데이터 크롤링
- 카카오 결제 api
