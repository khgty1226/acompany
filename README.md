# ACompany
: 고객이 되는 회사에게 제공되는 사이트로 회사 내부 사원을 위한 5가지 기능 '카풀 서비스', '사원 실적 통계', '마일리지 샵', '게시판', '채팅'을 제공하는 웹 사이트입니다.

사원들간의 소통 & 연결을 독려해주는 역할을 하며, 업무 실적 통계를 시각화하여 사원의 실적이 회사에 기여하는 정도를 확인하고 동기부여 할 수 있습니다.

업무, 소통에 적극적인 사원에게는 마일리지를 부여하여 마일리지샵에서 사무용품 등을 구매할 수 있습니다.

<br>

## 담당 기능
### 1. 카풀 (Carpool)
: 사원들간의 출/퇴근 카풀 목록을 등록/조회/신청/관리 할 수 있습니다

 
### 주요화면
- CarPool 목록 / 신청
- CarPool 등록
- 나의 CarPool
- 운행 기록 관리

### 2. 마일리지샵 (MileageShop)
: 사내 마일리지를 사용해 사무용품을 구매할 수 있는 쇼핑몰 기능입니다.


### 주요화면
- 상품 목록 조회
- 상품 상세정보 조회
- 장바구니
- 결제
